Projekte

1. Interaktive Wetter-App

Diese App zeigt aktuelle Wetterdaten an und nutzt eine Wetter-API, um die Informationen zu erhalten. Sie wurde mit HTML, CSS und JavaScript entwickelt und bietet eine benutzerfreundliche Oberfläche.

Skills: Webentwicklung, API-Integration, Front-End-Design

GitHub Repository: Wetter-App Code

2. IoT-basierte Smart Home App

Dieses Projekt ermöglicht die Steuerung von Smart Home-Geräten über eine mobile App. Es verwendet RESTful APIs und eine Datenbank zur Speicherung der Benutzereinstellungen und Geräteinformationen.

Skills: IoT, API-Integration, Mobile App Development, Backend Development

GitHub Repository: Smart Home Code

3. Energieverbrauchs-Tracker

Ein interaktives Dashboard zur Visualisierung von Energieverbrauchsdaten. Diese Anwendung verwendet JavaScript und D3.js, um die Daten in ansprechende Grafiken umzuwandeln.

Skills: JavaScript, Datenvisualisierung, D3.js, Front-End Development

GitHub Repository: Energieverbrauch Tracker Code

4. Personal Finance Manager

Ein Tool zur Verwaltung von persönlichen Finanzen. Es ermöglicht Benutzern, ihre Ausgaben zu verfolgen, Budgets zu erstellen und Sparziele zu setzen. Entwickelt mit HTML, CSS, JavaScript und einer API zur Berechnung von Finanzdaten.

Skills: Webentwicklung, API-Integration, JavaScript

GitHub Repository: Finance Manager Code
